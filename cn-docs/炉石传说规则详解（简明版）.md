# 炉石传说规则详解（简明版）

**版本**: 1.0  
**最后更新**: 2025-12-17  
**适用对象**: 开发者、设计师、高级玩家

---

## 前言

本文档是对《炉石传说》游戏规则的简化说明，旨在帮助读者快速理解游戏的核心机制。相比原版规则集，本文档更注重实用性和易理解性。

---

## 1. 游戏基础

### 1.1 游戏目标

**简单明了**: 将对手英雄的生命值降到0，你就赢了！

**特殊情况**:
- 双方同时死亡 → 平局
- 达到90回合 → 平局（疲劳伤害过高）
- 牌库抽空且疲劳伤害致死 → 输掉游戏

### 1.2 游戏组成

想象一场炉石对战就像两个魔法师在决斗：

- **英雄**: 你的角色，有30点生命值
- **英雄技能**: 每回合可以使用一次的特殊能力
- **套牌**: 30张卡牌组成
- **法力水晶**: 每回合可用的资源，用来支付卡牌费用

---

## 2. 回合流程

### 2.1 每个回合会发生什么？

**回合开始时**:
1. 你获得一个额外的法力水晶（最多10个）
2. 你的法力水晶完全恢复
3. 你可以再次使用英雄技能和攻击
4. 你从牌库抽一张牌

**你的回合中**:
- 可以使用手牌（需要支付费用）
- 可以让随从攻击（每回合一次）
- 可以使用英雄技能
- 可以什么都不做直接结束回合

**回合结束时**:
- 一些卡牌效果会触发（如回合结束扳机）
- 清除一些临时效果（如叫嚣的中士的+2攻击）

### 2.2 回合时间限制

- **正常回合**: 75秒
- **第一回合**: 更长一些（包含换牌时间）
- **90回合上限**: 超过90回合自动平局

---

## 3. 卡牌类型

### 3.1 随从（绿色边框）

**随从就像你的士兵**：
- 有攻击力和生命值
- 每回合可以攻击一次（除非有特殊效果）
- 可以留在战场上为你战斗
- 有各种各样的特殊能力

**随从的特殊能力**:
- **冲锋**: 入场就可以攻击
- **突袭**: 入场可以攻击随从，但不能攻击英雄
- **嘲讽**: 必须先攻击这个随从
- **潜行**: 不能被攻击或指定为目标
- **圣盾**: 第一次受到的伤害无效
- **剧毒**: 消灭任何被它伤害的随从
- **吸血**: 造成的伤害会治疗你的英雄
- **亡语**: 死亡时触发效果
- **战吼**: 使用时触发效果

### 3.2 法术（蓝色边框）

**法术是即时效果**：
- 使用后立即生效
- 然后进入墓地
- 只能在你的回合使用

**法术类型**:
- **火焰**: 通常是伤害法术
- **冰霜**: 通常是控制法术
- **自然**: 通常是治疗或增益
- **奥术**: 通常是抽牌或特殊效果
- **神圣**: 通常是治疗或保护
- **暗影**: 通常是伤害或削弱
- **邪能**: 恶魔相关的法术

### 3.3 武器（紫色边框）

**武器给你的英雄攻击力**：
- 装备后英雄可以攻击
- 有攻击力和耐久度
- 每次攻击消耗1点耐久度
- 耐久度为0时被销毁

**武器的特殊效果**:
- **风怒**: 每回合可以攻击两次
- **剧毒**: 消灭被攻击的随从
- **吸血**: 造成的伤害治疗英雄

### 3.4 英雄牌（橙色边框）

**英雄牌替换你的英雄**：
- 使用时立即获得新英雄
- 获得新的英雄技能
- 保留原有的生命值和护甲
- 触发战吼效果

## 4. 战场规则

### 4.1 随从位置

- **战场最多7个随从**
- 随从有固定的位置（1-7）
- 随从入场时占据最左边的空位
- 随从死亡后，右边的随从向左移动

### 4.2 随从攻击

**攻击规则**:
1. 随从入场当回合不能攻击（除非有冲锋/突袭）
2. 每回合只能攻击一次
3. 可以同时攻击随从或英雄
4. 攻击会同时对双方造成伤害

**攻击流程**:
1. 选择你的随从
2. 选择要攻击的目标
3. 双方同时造成伤害
4. 检查是否有随从死亡

### 4.3 特殊攻击情况

- **潜行**: 不能被攻击或指定
- **嘲讽**: 必须先攻击嘲讽随从
- **圣盾**: 第一次受到的伤害无效
- **免疫**: 完全不会受到伤害

## 5. 区域系统

想象你的卡牌可以在不同的地方移动：

### 5.1 区域类型

```
牌库 (DECK)          → 你的30张卡牌
手牌 (HAND)          → 你当前可使用的卡牌（上限10张）
战场 (PLAY)          → 已使用的随从、武器、奥秘
墓地 (GRAVEYARD)     → 已死亡的随从、已使用的法术
奥秘区 (SECRET)      → 已使用的奥秘和任务
除外区 (SETASIDE)    → 临时移动的区域
失效区 (REMOVED)     → 被移除的卡牌
```

### 5.2 区域移动规则

- **抽牌**: 牌库 → 手牌
- **使用**: 手牌 → 战场（随从/武器）或墓地（法术）或奥秘区（奥秘）
- **死亡**: 战场 → 墓地
- **弃牌**: 手牌 → 墓地
- **烧毁**: 抽牌时手牌已满 → 墓地

**重要规则**:
- 手牌最多10张，超过的会被烧毁
- 战场最多7个随从，超过的会被消灭
- 卡牌移动时会保留ID（是同一个实体）

## 6. 效果结算

### 6.1 效果顺序

**当一个效果发生时，按以下顺序处理**：

1. **预检测**: 检查是否能触发
2. **列队**: 收集所有响应的效果
3. **排序**: 按优先级和入场顺序排列
4. **结算**: 依次执行每个效果

**优先级规则**:
- 特殊优先级 > 普通优先级
- 场上 > 手牌 > 牌库
- 入场早的 > 入场晚的

### 6.2 常见优先级

**高优先级**（最先触发）:
- 回响消失
- 不稳定异变消失

**普通优先级**:
- 大多数扳机

**低优先级**（最后触发）:
- 救赎
- 吸血
- 超杀

**最低优先级**:
- 复生

### 6.3 嵌套结算

**当一个效果触发另一个效果时**：
- 先完成新效果
- 然后继续原效果
- 避免无限循环

**示例**: 北郡牧师 + 苦痛侍僧
1. 随从受到伤害
2. 苦痛侍僧触发，抽一张牌
3. 如果抽到治疗牌，北郡牧师触发
4. 继续原流程

## 7. 光环系统

### 7.1 什么是光环？

光环是持续影响其他卡牌的效果，比如：
- **暴风城勇士**: 你的其他随从+1/+1
- **鱼人领军**: 其他鱼人+2/+1
- **召唤传送门**: 你的随从-2费

### 7.2 光环的特点

- **持续生效**: 只要光环源在场就一直有效
- **实时更新**: 有新随从入场时会立即生效
- **低优先级**: 光环效果通常在最后计算
- **不复制**: 复制随从不会复制光环效果

### 7.3 光环优先级

**光环优先级规则**:
1. 特殊光环（如水晶核心）> 普通光环
2. 费用光环按添加顺序
3. 属性光环按添加顺序

**示例**: 小精灵 + 黑暗裁决 + 暴风城勇士
- 小精灵基础: 1/1
- 黑暗裁决: 3/3（状态）
- 暴风城勇士: 4/4（光环最后）

## 8. 死亡机制

### 8.1 什么是死亡？

**死亡条件**:
- 生命值 ≤ 0
- 被直接消灭（如刺杀）
- 武器耐久度 ≤ 0

**重要概念**: 濒死 ≠ 死亡
- 濒死: 被标记为待销毁
- 死亡: 真正从战场移除

### 8.2 死亡流程

**当一个随从濒死时**：
1. 标记为待销毁
2. 触发死亡时扳机
3. 从战场移除
4. 移动到墓地
5. 触发亡语效果

**特殊规则**:
- 濒死随从仍被视为在场上
- 可以被救回（如治疗）
- 沉默不能解除濒死状态

### 8.3 死亡阶段

**死亡阶段是特殊的处理时机**：
- 所有濒死实体一起移除
- 按入场顺序触发亡语
- 可能触发新的死亡阶段

**示例**: 自爆绵羊 + 机械克苏恩
1. 自爆绵羊死亡
2. 触发亡语，伤害其他随从
3. 机械克苏恩死亡
4. 触发亡语，消灭对手英雄

## 9. 特殊关键词

### 9.1 常见关键词解释

**冲锋 (Charge)**:
- 随从入场就可以攻击
- 可以攻击英雄或随从

**突袭 (Rush)**:
- 随从入场就可以攻击
- 只能攻击随从，不能攻击英雄

**剧毒 (Poisonous)**:
- 消灭任何被伤害的随从
- 即使只造成1点伤害

**吸血 (Lifesteal)**:
- 造成的伤害会治疗你的英雄
- 治疗量 = 造成的伤害

**圣盾 (Divine Shield)**:
- 第一次受到的伤害无效
- 被移除后不会恢复

**潜行 (Stealth)**:
- 不能被攻击或指定为目标
- 攻击后会失去潜行

**嘲讽 (Taunt)**:
- 必须先被攻击
- 对手不能绕过嘲讽攻击其他目标

**亡语 (Deathrattle)**:
- 随从死亡时触发效果
- 即使被沉默也会触发

**战吼 (Battlecry)**:
- 随从使用时触发效果
- 被复制的随从不会触发战吼

**连击 (Combo)**:
- 本回合已使用其他牌时触发
- 提供额外效果

**抉择 (Choose One)**:
- 使用时选择一种效果
- 某些卡牌可以触发两种效果

**回响 (Echo)**:
- 本回合可以再次使用
- 回合结束时消失

**复生 (Reborn)**:
- 第一次死亡时以1生命值复活
- 带有召唤失调

**超杀 (Overkill)**:
- 造成过量伤害时触发额外效果
- 只对随从有效

**法术伤害 (Spell Damage)**:
- 增加法术造成的伤害
- +1表示额外造成1点伤害

## 10. 常见交互场景

### 10.1 攻击场景

**场景1: 潜行随从攻击**
- 随从攻击后会失去潜行
- 可以被指定为法术目标

**场景2: 圣盾随从攻击**
- 攻击时圣盾不会被移除
- 被攻击时会移除圣盾

**场景3: 剧毒随从攻击**
- 即使造成0点伤害也会消灭目标
- 对英雄无效

### 10.2 法术场景

**场景1: 指向性法术**
- 需要选择目标
- 潜行随从不能被选为目标

**场景2: AOE法术**
- 影响所有随从
- 潜行随从不受影响

**场景3: 法术反制**
- 可以反制对手的法术
- 被反制的法术不会生效

### 10.3 亡语场景

**场景1: 多个亡语同时触发**
- 按入场顺序触发
- 救赎最后触发

**场景2: 亡语召唤新随从**
- 新随从入场
- 可能触发其他效果

**场景3: 亡语消灭其他随从**
- 触发新的死亡阶段
- 继续处理新的亡语

## 11. 游戏策略提示

### 11.1 回合规划

**每回合要考虑**：
1. 法力水晶使用情况
2. 手牌资源管理
3. 战场控制
4. 对手可能的应对

### 11.2 卡牌选择

**使用卡牌时要考虑**：
- 时机是否合适
- 是否有更好的选择
- 对手是否有反制手段
- 长期资源规划

### 11.3 风险管理

**高风险高回报的操作**：
- 使用随机效果
- 依赖特定组合
- 透支资源

## 12. 常见误区

### 12.1 新手容易犯的错误

❌ **误区1**: "濒死就是死亡"
✅ **正确**: 濒死不等于死亡，可以被救回

❌ **误区2**: "光环是永久性的"
✅ **正确**: 光环源离场后效果消失

❌ **误区3**: "潜行随从不能被攻击"
✅ **正确**: 潜行随从可以被AOE伤害

❌ **误区4**: "战吼可以被沉默"
✅ **正确**: 战吼是使用时效果，不能被沉默

❌ **误区5**: "复制的随从会触发战吼"
✅ **正确**: 复制的随从不会触发战吼

### 12.2 进阶玩家常遇到的问题

❌ **误区6**: "所有效果都立即生效"
✅ **正确**: 很多效果在阶段结束时才生效

❌ **误区7**: "入场顺序不重要"
✅ **正确**: 入场顺序决定扳机触发顺序

❌ **误区8**: "光环优先级高于状态"
✅ **正确**: 光环优先级低于状态效果

## 13. 实用技巧

### 13.1 记忆要点

**记住这些关键数字**：
- 手牌上限: 10张
- 随从上限: 7个
- 法力上限: 10个
- 生命值上限: 无上限（但显示上限为60）
- 回合上限: 90回合

### 13.2 快速判断

**如何快速判断效果顺序**：
1. 看是否有特殊优先级
2. 看所在区域
3. 看入场时间
4. 看控制器

### 13.3 常见组合

**经典组合效果**：
- 北郡牧师 + 治疗之环 = 大量抽牌
- 奥金尼灵魂祭司 + 治疗 = 造成伤害
- 先知维伦 + 伤害法术 = 伤害翻倍
- 瑞文戴尔男爵 + 亡语 = 亡语触发两次

## 总结

炉石传说的规则虽然复杂，但核心概念很简单：

1. **回合制**: 轮流行动
2. **资源管理**: 合理使用法力水晶和手牌
3. **效果响应**: 理解各种效果的触发时机
4. **策略思考**: 预测对手的行动

掌握这些基础规则后，你就可以享受这款深度策略游戏的乐趣了！
