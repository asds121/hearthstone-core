## 1. 什么是炉石传说？

炉石传说是一款**回合制卡牌游戏**，就像两个魔法师在决斗：

- 🧙‍♂️ **你是魔法师**（英雄），有30点生命值
- 🃏 **你有一副牌**（30张卡牌）
- ⚡ **你用法力水晶**来施放卡牌
- 🎯 **目标**是将对手的生命值降到0

### 1.1 游戏特色

- **简单易学**: 基本规则10分钟就能学会
- **深度策略**: 高级玩法需要精心规划
- **丰富内容**: 数千张卡牌，无数种组合
- **公平竞技**: 技术和策略决定胜负

## 2. 快速开始

### 2.1 游戏目标

**简单明了**: 把对手的生命值从30打到0，你就赢了！

### 2.2 基本流程

1. **每回合开始**: 你获得1个法力水晶（最多10个）
2. **抽一张牌**: 从牌库抽一张牌到手牌
3. **使用卡牌**: 用手牌和法力水晶进行操作
4. **攻击**: 让随从或英雄攻击对手
5. **结束回合**: 等待对手行动

### 2.3 你的第一个回合

想象这个场景：

```
回合1：
✓ 你获得1个法力水晶
✓ 你抽了一张牌
✓ 你手上有4张牌（1费、2费、2费、3费）
✓ 你可以使用1费的牌
✓ 然后结束回合
```

## 3. 认识卡牌

### 3.1 卡牌类型

炉石传说有4种卡牌，每种都有不同的颜色和作用：

#### 🟢 随从牌（绿色）

**你的士兵**
- 有攻击力和生命值
- 可以留在战场上战斗
- 每回合可以攻击一次

**示例**: 血沼迅猛龙
- 费用: 2
- 攻击力: 3 ⚔️
- 生命值: 2 ❤️

#### 🔵 法术牌（蓝色）

**一次性魔法**
- 使用后立即生效
- 然后消失（进入墓地）
- 有各种各样的效果

**示例**: 火球术
- 费用: 4
- 效果: 造成6点伤害

#### 🟣 武器牌（紫色）

**英雄的装备**
- 给英雄增加攻击力
- 有耐久度（使用次数）
- 耐久度用完就消失

**示例**: 炽炎战斧
- 费用: 2
- 攻击力: 3 ⚔️
- 耐久度: 2

#### 🟠 英雄牌（橙色）

**变身卡**
- 替换你的英雄
- 获得新的英雄技能
- 触发强力效果

**示例**: 死亡骑士牌
- 费用: 变化
- 效果: 变身并获得新技能

### 3.2 卡牌关键词

卡牌上有一些特殊的关键词，这些是游戏规则的重要组成部分：

#### ⚡ 冲锋
- 随从入场就可以攻击
- 非常适合快速压制

**示例**: 狼骑兵
- 2费 3/1 冲锋
- 入场就能打3点伤害

#### 🏃 突袭
- 随从入场可以攻击随从
- 但不能攻击英雄
- 适合解场

**示例**: 狂奔科多兽
- 3费 3/5 突袭
- 入场就能清掉对手的随从

#### 🛡️ 嘲讽
- 必须先攻击这个随从
- 保护其他随从和英雄
- 非常适合防守

**示例**: 银背族长
- 3费 1/4 嘲讽
- 对手必须先打他

#### 👻 潜行
- 不能被攻击或指定为目标
- 只有AOE法术能打到
- 适合偷袭

**示例**: 丛林猎豹
- 3费 4/2 潜行
- 对手无法直接攻击他

#### ✨ 圣盾
- 第一次受到的伤害无效
- 相当于有一次免伤机会
- 非常实用

**示例**: 银色侍从
- 1费 1/1 圣盾
- 性价比极高

#### ☠️ 剧毒
- 消灭任何被伤害的随从
- 无论伤害多少
- 解场神技

**示例**: 帝王眼镜蛇
- 3费 2/3 剧毒
- 能消灭任何大随从

#### 💉 吸血
- 造成的伤害会治疗你的英雄
- 治疗量 = 造成的伤害
- 续航能力强

**示例**: 鲜血掠夺者古尔丹
- 10费 很多效果 吸血
- 大量回血

#### 💀 亡语
- 随从死亡时触发效果
- 即使被沉默也会触发
- 死亡也有价值

**示例**: 战利品贮藏者
- 2费 2/1 亡语：抽一张牌
- 死了还能过牌

#### 📢 战吼
- 使用时触发效果
- 被复制的随从不会触发
- 入场效果

**示例**: 奥术智慧
- 3费 战吼：抽两张牌
- 过牌神器

## 4. 游戏流程详解

### 4.1 回合结构

每个回合都有固定的流程：

```
┌─────────────────────────────┐
│      你的回合开始了！        │
├─────────────────────────────┤
│ 1. 获得一个法力水晶          │
│    （最多10个）              │
│ 2. 法力水晶完全恢复          │
│ 3. 抽一张牌                  │
│ 4. 可以再次攻击和使用技能    │
└─────────────────────────────┘
         ↓
┌─────────────────────────────┐
│      你的行动阶段            │
├─────────────────────────────┤
│ • 使用手牌                   │
│ • 让随从攻击                 │
│ • 使用英雄技能               │
│ • 什么都不做                │
└─────────────────────────────┘
         ↓
┌─────────────────────────────┐
│      回合结束阶段            │
├─────────────────────────────┤
│ • 触发回合结束效果           │
│ • 清除临时效果               │
│ • 切换到对手回合             │
└─────────────────────────────┘
```

### 4.2 使用卡牌

#### 4.2.1 随从使用流程

1. **选择随从**: 从手牌中选择要使用的随从
2. **支付费用**: 消耗相应的法力水晶
3. **随从入场**: 随从进入战场
4. **触发战吼**: 如果有战吼，立即触发
5. **触发扳机**: 其他卡牌可能会响应

**注意**: 随从入场当回合通常不能攻击（除非有冲锋或突袭）

#### 4.2.2 法术使用流程

1. **选择法术**: 从手牌中选择要使用的法术
2. **选择目标**: 如果需要目标，选择目标
3. **支付费用**: 消耗相应的法力水晶
4. **法术生效**: 法术效果立即生效
5. **进入墓地**: 法术使用后进入墓地

**注意**: 法术反制可以阻止法术生效

#### 4.2.3 武器使用流程

1. **选择武器**: 从手牌中选择要使用的武器
2. **支付费用**: 消耗相应的法力水晶
3. **装备武器**: 武器装备到英雄
4. **触发战吼**: 如果有战吼，立即触发
5. **旧武器销毁**: 如果有旧武器，会被销毁

**注意**: 武器给英雄提供攻击力

### 4.3 战斗系统

#### 4.3.1 攻击规则

**谁可以攻击？**
- 有攻击力的随从
- 装备武器的英雄
- 攻击力 > 0
- 没有召唤失调（入场当回合）

**可以攻击谁？**
- 对手的随从
- 对手的英雄
- 如果有嘲讽随从，必须先攻击嘲讽随从

#### 4.3.2 伤害计算

**基础伤害** = 攻击者的攻击力

**特殊情况**:
- 圣盾: 第一次受到的伤害无效
- 免疫: 完全不会受到伤害
- 潜行: 不能被攻击

#### 4.3.3 攻击流程

1. **选择攻击者**: 选择一个可以攻击的随从或英雄
2. **选择目标**: 选择一个合法的攻击目标
3. **造成伤害**: 双方同时造成伤害
4. **检查死亡**: 检查是否有随从死亡
5. **触发效果**: 触发攻击相关的效果

**示例**: 
- 你的3/2随从攻击对手的2/3随从
- 你的随从造成3点伤害，对手的随从造成2点伤害
- 两个随从都会受到伤害

## 5. 区域系统

### 5.1 区域介绍

想象你的卡牌可以在不同的地方移动：

#### 📚 牌库（Deck）
- 你的30张卡牌
- 游戏开始时在这里
- 每回合从这里抽牌

#### 🤲 手牌（Hand）
- 你当前可以使用的卡牌
- 最多10张
- 超过10张会被烧毁

#### ⚔️ 战场（Play）
- 已使用的随从和武器
- 随从最多7个
- 超过7个会被消灭

#### 💀 墓地（Graveyard）
- 死亡的随从
- 已使用的法术
- 被弃掉的卡牌

#### 🔮 奥秘区（Secret）
- 已使用的奥秘
- 对手看不到
- 触发条件后生效

#### 🗂️ 除外区（Setaside）
- 临时移动的区域
- 发现、追踪等效果使用

### 5.2 区域移动

卡牌可以在区域间移动：

```
牌库 → 手牌 → 战场
  ↓      ↓      ↓
墓地 ← 除外 ← 奥秘
```

**常见移动**:
- 抽牌: 牌库 → 手牌
- 使用: 手牌 → 战场/墓地/奥秘
- 死亡: 战场 → 墓地
- 弃牌: 手牌 → 墓地
- 烧毁: 牌库 → 墓地

## 6. 特殊机制

### 6.1 死亡机制

#### 6.1.1 什么是死亡？

**死亡条件**:
- 生命值 ≤ 0
- 被直接消灭
- 武器耐久度 ≤ 0

**重要概念**: 
- **濒死**: 被标记为待销毁，但还在场上
- **死亡**: 真正从战场移除

#### 6.1.2 死亡流程

1. **标记濒死**: 生命值降到0或更低
2. **触发效果**: 死亡时扳机触发
3. **移除实体**: 从战场移除
4. **触发亡语**: 亡语效果触发

**注意**: 濒死随从仍被视为在场上，可以被救回

### 6.2 光环机制

#### 6.2.1 什么是光环？

光环是持续影响其他卡牌的效果：

**常见光环**:
- 暴风城勇士: 其他随从+1/+1
- 鱼人领军: 其他鱼人+2/+1
- 召唤传送门: 随从-2费

#### 6.2.2 光环特点

- **持续生效**: 只要在场上就一直有效
- **实时更新**: 新随从入场立即生效
- **低优先级**: 最后计算
- **不复制**: 复制随从不复制光环

### 6.3 随机性

#### 6.3.1 随机效果

很多卡牌有随机效果：
- 随机目标
- 随机伤害
- 随机抽牌
- 随机召唤

#### 6.3.2 随机规则

- 伤害类效果：不会选择濒死目标
- 治疗类效果：不会选择满血目标
- 抽牌类效果：可能抽到烧毁
- 召唤类效果：可能召唤失败

## 7. 常见情况处理

### 7.1 满手牌

**当你有10张手牌时**：
- 再抽牌会烧毁（直接进入墓地）
- 获得卡牌也会烧毁
- 注意资源管理

### 7.2 满战场

**当战场有7个随从时**：
- 不能再召唤新随从
- 如果强制召唤，新随从会被消灭
- 注意场面控制

### 7.3 同时死亡

**当多个随从同时死亡时**：
- 按入场顺序移除
- 按入场顺序触发亡语
- 可能触发连锁反应

### 7.4 平局

**游戏可能平局的情况**：
- 双方英雄同时死亡
- 达到90回合上限
- 双方都投降

## 8. 实用技巧

### 8.1 新手建议

#### ✅ 应该做的

1. **先理解基础规则**
   - 了解每种卡牌的作用
   - 熟悉回合流程
   - 掌握基本操作

2. **从简单卡组开始**
   - 使用基础卡组
   - 逐步添加新卡牌
   - 不要急于求成

3. **注意资源管理**
   - 合理使用法力水晶
   - 不要浪费手牌
   - 规划长期策略

4. **学习卡牌组合**
   - 了解卡牌间的配合
   - 尝试不同组合
   - 发现强力combo

5. **观察对手**
   - 注意对手的手牌数
   - 预测对手的行动
   - 制定应对策略

#### ❌ 不应该做的

1. **不要急于求成**
   - 炉石需要慢慢学习
   - 失败是正常的
   - 从失败中学习

2. **不要盲目使用卡牌**
   - 考虑时机和目标
   - 不要浪费资源
   - 思考最优解

3. **不要忽视场面**
   - 随从很重要
   - 控制场面是胜利关键
   - 不要只打脸

4. **不要忽视对手**
   - 对手也有策略
   - 要预测对手行动
   - 做好应对准备

### 8.2 进阶技巧

#### 8.2.1 资源交换

学会计算资源价值：
- 1费随从 ≈ 1.5价值
- 2费随从 ≈ 2.5价值
- 3费随从 ≈ 3.5价值

**目标**: 用你的低价值卡牌换取对手的高价值卡牌

#### 8.2.2 节奏控制

理解游戏节奏：
- **前期**（1-3费）: 建立场面
- **中期**（4-6费）: 扩大优势
- **后期**（7+费）: 决胜阶段

**目标**: 在每个阶段都做正确的事

#### 8.2.3 手牌管理

学会管理手牌：
- 保持手牌数量
- 不要过度使用
- 保留关键卡牌

**目标**: 保持手牌优势

## 9. 常见误区

### 9.1 新手常见错误

❌ **误区1**: "濒死就是死亡"
✅ **正确**: 濒死不等于死亡，可以被救回

❌ **误区2**: "光环是永久性的"
✅ **正确**: 光环源离场后效果消失

❌ **误区3**: "潜行随从不能被攻击"
✅ **正确**: 潜行随从可以被AOE伤害

❌ **误区4**: "战吼可以被沉默"
✅ **正确**: 战吼是使用时效果，不能被沉默

❌ **误区5**: "复制的随从会触发战吼"
✅ **正确**: 复制的随从不会触发战吼

### 9.2 进阶误区

❌ **误区6**: "所有效果都立即生效"
✅ **正确**: 很多效果在阶段结束时才生效

❌ **误区7**: "入场顺序不重要"
✅ **正确**: 入场顺序决定扳机触发顺序

❌ **误区8**: "光环优先级高于状态"
✅ **正确**: 光环优先级低于状态效果

## 12. 常见问题解答

### Q1: 我应该从哪个职业开始？

**A**: 推荐从法师或猎人开始：
- **法师**: 法术丰富，容易理解
- **猎人**: 节奏明快，适合新手

### Q2: 如何获得新卡牌？

**A**: 多种方式：
- 完成新手任务
- 购买卡包
- 参与活动
- 制作卡牌

### Q3: 什么是元游戏？

**A**: 当前流行的卡组和策略：
- 了解主流卡组
- 学习应对策略
- 适时调整卡组

### Q4: 如何提高胜率？

**A**: 多练习，多思考：
- 理解自己的卡组
- 了解对手的卡组
- 做出正确决策

### Q5: 遇到bug怎么办？

**A**: 及时反馈：
- 记录详细信息
- 截图或录像
- 联系客服

## 13. 游戏规则速查

### 13.1 关键数字

| 项目 | 数值 | 说明 |
|------|------|------|
| 英雄生命值 | 30 | 起始生命值 |
| 手牌上限 | 10 | 超过会烧毁 |
| 随从上限 | 7 | 超过会被消灭 |
| 法力上限 | 10 | 最多10个 |
| 回合上限 | 90 | 超过自动平局 |

### 13.2 优先级顺序

1. 特殊优先级（回响消失等）
2. 普通优先级（大多数效果）
3. 低优先级（救赎、吸血等）
4. 最低优先级（复生）

### 13.3 区域移动

- 牌库 → 手牌: 抽牌
- 手牌 → 战场: 使用随从/武器
- 手牌 → 墓地: 使用法术/弃牌
- 战场 → 墓地: 死亡
- 手牌/牌库 → 墓地: 烧毁
